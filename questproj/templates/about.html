technical details

Django and Tastypie are used to serve the Clue data.

The database is PostgreSQL.

Raw clue data gathered and parsed with BeautifulSoup.

Uses modularized Javascript components via Node.js, built with Browserify.

The Javascript app framework is Backbone.js


api

A RESTful api interface to the clues is rendered with Tastypie and Backbone-tastypie.
